# InfoPC
Script en powershell que genera un csv y recopila la siguiente informacion del equipo:

- Hostname
- Dominio
- Sistema Operativo
- Fecha Instalacion OS
- Usuario Actual
- Total Usuarios
- Tipo de Equipo
- Fabricante
- Modelo
- Numero de Serie
- CPU
- GPU
- RAM (GB)
- Unidades de almacenamiento
- Unidades de red
- NIC
- Pantallas
- Impresoras
- Fecha datos
